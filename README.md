# waybar
Ce dépôt contient ma configuration personnelle pour Waybar, optimisée pour mon environnement Hyprland. Elle inclut les fichiers de style et de disposition pour obtenir une barre de statut à la fois esthétique et fonctionnelle. Pour l'utiliser, il suffit de cloner ces fichiers dans votre répertoire ~/.config/waybar/
